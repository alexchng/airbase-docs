# Templates

Templates are quick and easy ways to get started building an application!

Whether it’s prototyping an idea, building a proof-of-concept, or laying the foundation for the next big thing, there’s a template for you.

## Where to get templates

| Name                  | Type                 | Link                                                                                                                   |
| --------------------- | -------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| airbase-t3-app        | Full Stack - NextJS  | [SGTS Innersource](https://sgts.gitlab-dedicated.com/innersource/sgts/runtime/airbase/templates/airbase-t3-app)        |
| airbase-streamlit-app | Full Stack - Python3 | [SGTS Innersource](https://sgts.gitlab-dedicated.com/innersource/sgts/runtime/airbase/templates/airbase-streamlit-app) |
| airbase-go-app        | Backend API - Go     | [SGTS Innersource](https://sgts.gitlab-dedicated.com/innersource/sgts/runtime/airbase/templates/airbase-go-app)        |

## Making your own template

Can’t find something that catches your eye? Feel free to create your own from scratch.

If you’d like to contribute it later, [let us know](https://go.gov.sg/airbase-feedback).
