# Logs

The Airbase Console makes it easy to manage and operate projects.

Viewing the logs of a project running on Airbase is a common task that developers need to do.

![Screenshot of the Logs Viewer in the Airbase Console](https://console.v2.airbase.sg/docs/_next/image?url=%2Fdocs%2F_next%2Fstatic%2Fmedia%2Fairbase-console-logs.d47a9f0c.png\&w=3840\&q=75)

## Instructions

Here’s how to view the logs for a project.

### View Project Logs

Navigate to the project in the [Airbase Console](https://console.v2.airbase.sg/projects), and click “Logs” in the side navigation bar.

### Select Environment

Select the environment you’re interested in from the dropdown menu in the logs pane.
