# Airbase Console

The Airbase Console is the one-stop-shop for everything you need.

## Managing Projects with Airbase

![Screenshot of the Airbase Overview page](https://console.v2.airbase.sg/docs/_next/image?url=%2Fdocs%2F_next%2Fstatic%2Fmedia%2Fairbase-console-overview.f4b60d79.png\&w=3840\&q=75)

In the Projects Overview, developers can easily check on the status of their application, make changes, and keep an eye on incoming logs.

[Viewing Logs](logs.md)

## Administration

It’s also where you can create teams, manage projects, and rotate API credentials.

* [Create Teams](https://console.v2.airbase.sg/settings/teams)
* [Manage Projects](https://console.v2.airbase.sg/projects)
* [Rotate API credentials](https://console.v2.airbase.sg/settings/credentials)
