# Introduction to Airbase

## Introduction to Airbase

Airbase is the fastest, easiest way to deploy web applications to GCC.

It allows teams to rapidly build and launch new applications.

### Get Started

Looking for a way to prototype an idea, or build a proof-of-concept?

This quickstart guide walks through setting up and deploying your first project.

2. [Installation](get-started/installation.md)
